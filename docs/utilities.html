Responsive prefixes

Mobile-first breakpoints. Prefix any layout utility: sm: 640px, md: 768px, lg: 1024px, xl: 1280px.

<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-4"> … </div>
